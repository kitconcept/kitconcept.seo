SEO enhancements for the Plone Content Management System.

Features
--------

- Adds SEO fieldset to Dexterity content types to override title and description for SEO

--------

------------

Install kitconcept.seo by adding it to your buildout::

    [buildout]

    ...

        kitconcept.seo
